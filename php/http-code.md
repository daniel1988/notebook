
Http 状态码

## 1** 信息提示

* 100 初始的请求已接受

* 101 switching protocols

## 2开头（请求成功）表示成功处理了请求的状态码

* 200 请求成功

* 201 已创建

* 202 已接受

## 3开头(请求重定向)表示要完成请求，需要进一步操作．




## 4开头(客户端请求错误)

* 400 错误请求

* 401 未授权

* 403 服务器拒绝请求

* 404 服务器找不到请求


## 5开头(服务端错误)

* 500 服务器内部错误

* 501 尚未实施

* 502 错误网关

* 503 服务不可用

* 504 网关超时
